# House Rule Stickers
Permanent effect for the rest of the run.
Resets in shop every 2 rounds.
Sticker 1 shows up in shop 1 & 2. Sticker 2 shows up in shop 3 & 4, etc.

- Extra ball: +1 ball
- Extra ring: +1 scoring area
- Bigger Holes: All scoring areas size increased by 10%
- Golden Ticket: 25% more ticket rewards (rounded up)
- Extra Totem: +1 totem slot
- Extra Stunt: +1 stunt slot
- Extended Warranty: Increased all item sell values
- Quick Hands: 2 stunts can be used on one throw
- More balls: ball upgrades are more likely to appear in the shop
- More stunts: stunt upgrades are more likely to appear in the shop
- More engineering: engineering upgrades are more likely to appear in the shop
