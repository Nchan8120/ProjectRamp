# ProjectRamp
Skee-ball style Roguelike

# General Gameplay
- Player aims and throws a physics based ball up the ramp towards a series of rings that direct the ball into holes of varying point values. The hardest to reach holes are worth the most points.
- Player gets a limited amount of throws.
- If the player scores below a certain number of points they get a game over.
- If the player scores above the point threshold they get to go a shop.
- In the shop they can purchase powerups.
- Then they progress to the next round where the point requirement goes up.
